Dithering generator
=============

Generate dithering effect


Code example
-----------
```javascript
var dg = new DitherGen(canvas);
var imgData = dg.createGradient(true, [255, 0, 0, 255], [0, 0, 255, 255]);
dg.dither(imgData);
```
